# AI Game Company Server v1

## 프로젝트 개요
이 서버는 AI 지원 소프트웨어 개발 워크플로우를 관리하는 제어 평면(Control Plane) 프로그램입니다.

## 이 서버가 하는 일
- **프로젝트 및 작업 계획 관리**: 에픽, 서브에픽, 태스크 계층 구조 관리.
- **작업자 실행 제어**: Worker 프로세스의 작업 대여(Lease), 실행 및 결과 보고 관리.
- **검증 및 아티팩트 관리**: 테스트 러너(Test Runner) 실행 결과 검증, 빌드 및 테스트 로그/스크린샷 등의 아티팩트(Artifact) 보관.
- **결재 및 병합 흐름**: 오너 AI의 작업물 검토, 자연어 승인 결재 흐름, 프로젝트 브랜치 머지 API 처리.
- **장기 프로젝트 메모리**: 개발 의사결정 및 작업 이력의 저장 및 태그 기반 컨텍스트 검색.

## 이 서버가 하지 않는 일
- **게임 런타임 서버가 아님**: 실시간 게임 플레이 데이터를 동기화하거나 멀티플레이를 처리하는 게임 서버가 아니라, 오직 개발 공정 자동화를 제어하는 제어 도구입니다.

## 현재 상태
- **Golden Path 리허설 통과**: 가상의 저장소와 워크스페이스를 사용해 태스크 대여, 임시 코드 수정, 브랜치 푸시, 테스트 러너 검증 및 리포트 작성, 오너 머지 승인 결재까지 E2E 전체 자동화 루프 작동 확인이 완료되었습니다.

## 보안 주의사항
> [!WARNING]
> - **포트 노출 금지**: HTTPS 역프록시, 토큰 인증 설정, 또는 사설 VPN(Tailscale 등) 보호 조치 없이 외부 오픈 인터넷에 본 서버의 raw 포트(예: `:8080`)를 직접 노출하지 마십시오.
> - **자격 증명 노출 금지**: 데이터베이스나 파일 내에 실제 API 키, 토큰, 비밀번호 등을 하드코딩하거나 깃에 커밋하지 마십시오. `.env` 파일은 절대로 리포지토리에 추적/커밋하지 않습니다.
